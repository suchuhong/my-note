    sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

    # ssh 连接

    要在 VM 虚拟机中的 Ubuntu 上启用 SSH，可以按照以下步骤进行操作：

    1. 安装 SSH 服务器 在 Ubuntu 终端中，使用以下命令安装 SSH 服务器：
    sudo apt-get update
sudo apt install ssh
sudo apt-get install openssh-server
    1
    2
    3
    1. 配置 SSH 服务器 安装完成后，使用以下命令打开 SSH 服务器配置文件：
    sudo nano /etc/ssh/sshd_config
    1

    在文件中找到以下行：

    #Port 22
#AddressFamily any
#ListenAddress 0.0.0.0
#ListenAddress ::
    1
    2
    3
    4

    将其中的 #Port 22 的注释符号 # 去掉，并将端口号改为自己想要的端口号（例如 2222），如下所示：

    Port 2222
#AddressFamily any
#ListenAddress 0.0.0.0
#ListenAddress ::
    1
    2
    3
    4
    1. 重启 SSH 服务器 使用以下命令重启 SSH 服务器：
    
sudo ufw allow ssh

sudo service ssh restart
    1
    2
    3
    4
    5
    1. 查看是否启动
    sudo systemctl status ssh
    1
    1. 测试本机连接
    ssh username@ip_address
    1
    1. 测试局域网连接
    ssh username@ip_address
    1
